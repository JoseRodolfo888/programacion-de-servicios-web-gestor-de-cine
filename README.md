# programacion-de-servicios-web-gestor-de-cine
**Nuestro proyecto consta en un programa que administra una app de cine.**

Contiene usuario (crear cuenta, iniciar sesion, ver cartelera y comprar boletos de la pelicula a eleccion, productos para la sala de cine), administrador (agregar peliculas con sus caracteristicas, administrar salas, administrar las caracteristicas de las funciones y administrar los productos), empleados.
